# Assets 文件夹

请将 Eastern Market logo 图片保存到此文件夹。

## 支持的文件格式
- `eastern_market_logo.jpg` (推荐)
- `eastern_market_logo.jpeg` 
- `eastern_market_logo.png`

## Logo 要求
- 格式：JPEG 或 PNG
- 建议尺寸：400x200 像素或更高
- 背景：白色或浅色（JPEG 格式）
- 文件名：必须是上述三个名称之一

## 优先级
系统会按以下顺序查找 logo 文件：
1. `eastern_market_logo.jpg`
2. `eastern_market_logo.jpeg`  
3. `eastern_market_logo.png`

找到第一个存在的文件就会显示，如果都不存在则显示备用图标。

## 使用方式
Logo 将会显示在应用页面的顶部，宽度为 200px。
